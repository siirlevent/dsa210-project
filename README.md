# dsa210-project
Term project for DSA210

This repository contains my term project for the course **DSA210 (Spring 2026)**.

## Project Idea
The goal of this project is to analyze a real-world dataset and apply data science techniques including exploratory data analysis, visualization, and machine learning.

## Planned Steps
1. Data collection
2. Data cleaning and preprocessing
3. Exploratory Data Analysis (EDA)
4. Statistical analysis and hypothesis testing
5. Machine learning models
6. Visualization and reporting

## Tools
Python, Pandas, NumPy, Matplotlib, Scikit-learn
